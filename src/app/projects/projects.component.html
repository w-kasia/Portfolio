01

Baczyński

It’s a website dedicated to Polish poet Krzysztof Kamil Baczyński. For this project, I utilized Angular and Angular Material to create a user-friendly interface. The app includes responsive design components.
One of the features is contact form. I implemented a template-driven form using Angular, and to store the data submitted through the form, I utilized Firebase. This ensures that the contact form submissions are securely stored and easily accessible.
To enrich the project further, I incorporated an open API to provide poet’s works.

Eat&Chill

It’s a restaurant website. To create that project, I’ve utilized the power of Angular and Bootstrap. I’ve implemented both reactive and template-driven form to enhance the user experience.
To store the data from forms securely, I’ve integrated Firebase into the project.
We’ve got some interesting features like gallery, menu and FAQ sections. The website is fully responsive. Customers can easily navigate through the website and access all information they need.
